# Mentora — Video Generation Prompt

Используй для AI-видео-инструментов: **HeyGen**, **Synthesia**, **Runway**, **Pika**, **Kling** или **Sora**.

---

## 🎬 Промт для AI-аватара (HeyGen / Synthesia)

**Язык:** Русский  
**Длительность:** 90–120 секунд  
**Стиль:** Молодёжный, живой, не занудный. Темп — быстрый, но чёткий.

### Скрипт для аватара

---

*[Аватар смотрит в камеру. Тёмный фон с лёгким синим свечением.]*

Привет! Ты открыл Mentora — AI-ментора, который реально помогает учиться.

Вот как это работает за 60 секунд.

*[Экранная запись: карточки предметов на дашборде]*

Выбираешь предмет. Сейчас их четырнадцать — от математики до астрономии. Нажимаешь — и открывается чат.

*[Экранная запись: ввод вопроса в чат]*

Пишешь вопрос как другу. Без шаблонов. Например: «почему Наполеон проиграл под Ватерлоо?» — и получаешь нормальный ответ, не статью из Википедии.

*[Аватар]*

Ментора помнит весь контекст разговора. Можно углубляться, переспрашивать, просить объяснить иначе. Он не устанет и не раздражится.

*[Экранная запись: шапка с мент-счётчиком и огонёк стрика]*

За каждый ответ — менты, это твой XP. Занимаешься каждый день — растёт стрик. Пропустил — сбросился. Это мотивирует.

*[Аватар]*

Базовый план — бесплатно. Двадцать сообщений в день. Pro — четыреста девяносто девять рублей в месяц. Безлимит, все предметы, память между сессиями.

И последнее: в профиле есть твоя реферальная ссылка. Пригласи друга — вам обоим плюс три дня Pro. Сразу. Бесплатно.

Попробуй прямо сейчас — mentora.su

## 🎨 Визуальный стиль для motion-graphics / Sora / Runway

```
Visual style brief for Mentora tutorial video:

Color palette:
- Background: deep dark #080814, almost black navy
- Primary accent: electric blue #4561E8
- Secondary: indigo #6366F1, violet #A78BFA  
- Text/highlights: pure white, warm white rgba(255,255,255,0.85)

Typography feel:
- Headlines: ultra-bold, tight letter spacing, modern sans-serif (Inter/Geist)

Visual elements:
- Glassmorphism panels: dark glass with blur, subtle white borders
- Blue ambient glow from top-left, violet from bottom-right
- Particle field background (subtle, slow-moving blue dots)
- 3D card hover effect: slight Y-rotation on card entry
- Smooth spring animations (not linear, physics-based)

UI to show:
1. Dashboard grid of subject cards (dark glass cards, colored emoji icons)
2. Chat interface (message bubbles with gradient AI responses)
3. Stats bar in nav (XP pill in blue, streak pill in orange)
4. Profile page with progress bars and badges

Transitions:
- Slide from right on new section, fade-scale on card appearance
- No hard cuts — always blend/crossfade

Mood: Intelligent, calm, slightly futuristic. Like a premium productivity app.
Not childish, not corporate. Target: Russian teenagers + young adults 15-25.
```

## 🎙️ Озвучка (ElevenLabs / Yandex SpeechKit)

**Голос:** Молодой мужской или женский, нейтральный русский акцент, уверенный темп.  
**Настройки ElevenLabs:** Stability 0.45, Similarity 0.75, Style 0.3  
**Темп речи:** 1.05x (чуть быстрее нормального)

**Альтернатива:** Яндекс SpeechKit — голос `alena` (женский) или `filipp` (мужской), скорость 1.1.
