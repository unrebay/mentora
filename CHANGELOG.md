# Changelog

Все значимые изменения Mentora. Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/); версии — [SemVer](https://semver.org/lang/ru/) (major.minor).

Пользовательская версия истории — на странице [«О нас» → «Что нового»](https://mentora.su/dashboard/about) (источник: `src/lib/changelog.ts`).

## [6.3.0] — 2026-06-04

### Добавлено
- ⚡ **Стриминг ответов ИИ**: чат и демо на лендинге печатают ответ токен-за-токеном (~1с до первых слов вместо ~6с ожидания). Сервер: `ReadableStream` + финальное META-событие с метаданными (XP, level-up, стрик, подсказки, лимиты, картинка, цитаты).
- Раздел **«Что нового»** на странице «О нас» — история версий из `src/lib/changelog.ts`.
- Клиентское сжатие фото для vision (canvas, до 1600px JPEG) — надёжнее и быстрее загрузка.
- Семафор параллельности Anthropic для стримов (`acquireAnthropicSlot`) — перегруз отдаёт аккуратный 429.

### Изменено
- **Главная страница — статическая** (prerender вместо force-dynamic SSR): ~4.5–6с → доли секунды. Редирект залогиненных перенесён в middleware.
- Vercel-прокси к Anthropic переведён на **Edge runtime** (Node-runtime буферизовал SSE); секрет `ANTHROPIC_BASE_URL` переключён со старого immutable deploy-URL на стабильный production-домен.
- Деплой: распаковка релиза до рестарта + prune только после health-check (окно рестарта — секунды вместо десятков секунд).

### Исправлено
- Ссылки «Поделиться прогрессом» были нерабочими всегда: UI строил URL из `id` вместо `token`, а API под RLS не отдавал данные гостям/чужим аккаунтам (теперь service-role, токен = пропуск).
- Google-вход «отбрасывал на форму/лендинг» после деплоев (даунтайм-окно + сгоревший OAuth-state); добавлено самолечение на /auth и в колбэке.
- Индикатор «три точки» больше не висит поверх печатающегося ответа.
- Ошибка стрима снова возвращает сообщение в free-лимит и шлёт алерт админу; таймаут стрима 120с (длинные vision-ответы не обрываются).
- Edge-функция `embed` закрыта авторизацией (verify_jwt).

## [6.2.0] — 2026-06-01

### Добавлено
- 🔐 **Вход по passkey** (Face ID / отпечаток / Windows Hello) — регистрация в Профиле, вход на /auth.
- **Стрик-сейвер**: вечернее напоминание при риске потери стрика; **Ultra — авто-заморозка стрика**.
- **Годовой план** с промо «+3 месяца» (единый конфиг `src/lib/promo.ts`).
- Чекбокс согласия на автопродление на чекауте (default on) — рекуррент реально продлевает подписку.

### Изменено
- Минимизация данных: поле «телефон» удалено из продукта и БД (152-ФЗ).
- Формулировка о защите данных на лендинге обновлена (решение Andy).

### Исправлено
- `auto_renew` не включался после покупки → подписки не продлевались.
- Бенчмарк точности: ретраи 529/справедливый судья → честные 97% (claim на лендинге подтверждён).
- Service-worker кэш бастится по git SHA каждый деплой — «вернувшиеся» пользователи больше не ловят сломанные страницы.
